# 実装中の計算式
## IIDX

$Total = \max\left(260, \frac{7.605 * note}{0.01 * note + 6.5}\right)$

## pop'n

$Total = \frac{note * \lfloor \frac{3072}{note} \rfloor}{10.24}$

## beatmania

$Total = \frac{note * \lfloor \frac{fix}{note} \rfloor}{55}$

fix値は以下の通り。
- FINAL: 14933
- 7th: 14000
- core: 16800
- comp2: 12133
- 5th: 13066
- 4th: 20533
